AssetRender
===========
Unity headless application contained in a Docker system.
 
Toolchain
---------
The following applications are required for development.

**Windows**
  - Unity 4.6.7p4
  - Visual Studio Code
  - Git 1.8+
  - SourceTree
  - Ant 1.9.7+ (for automated building)

**OSX**
  - Unity 4.6.7p4
  - Visual Studio Code
  - Git 1.8+
  - Ant 1.9.7+ (for automated building)
